NutriScan API

Our API allows you to integrate NutriScan's nutrition analysis into your own applications.

API Endpoint

POST https://api.calorieninjas.com/v1/imagetextnutrition

Upload an image of food to get nutritional information.

Example Request

curl -X POST "https://api.calorieninjas.com/v1/imagetextnutrition" \
-H "X-Api-Key: YOUR_API_KEY" \
-F "file=@path_to_your_image.jpg"

Response Example

{
  "items": [
    {
      "name": "Carrot",
      "calories": 41,
      "protein_g": 0.9,
      "carbohydrates_total_g": 10,
      "fat_total_g": 0.2,
      "sugar_g": 4.7,
      "fiber_g": 2.8
    }
  ]
}
